# IT3040 – ITPM Assignment 1: Transliteration Accuracy Testing

**Student Registration Number:** IT23278844  
**Git Repository:** https://github.com/D-Sachin/IT3040-Assignment1

---

## Overview

This project automates **50 negative test cases** for the **PixelsSuite Chat Translator** ([https://www.pixelssuite.com/chat-translator](https://www.pixelssuite.com/chat-translator)), which converts chat-style Singlish input into Sinhala output.

The test cases cover all **24 Singlish input types** defined in the assignment specifications, identifying scenarios where the translator fails to produce the correct Sinhala output. In a negative test suite, all 50 cases are designed to result in a **FAIL** status to demonstrate the system's limitations.

---

## Prerequisites

Ensure the following are installed on your system:

- **Python 3.11 or 3.12** — [https://www.python.org/downloads/](https://www.python.org/downloads/)
- **Google Chrome** (recommended)

---

## Setup Instructions

### Step 1 — Navigate to the project
Open **Command Prompt** or terminal and navigate to the `test_automation` directory:

```cmd
cd test_automation
```

### Step 2 — Install dependencies
Install the required Python libraries and Playwright browsers:

```cmd
pip install playwright openpyxl
playwright install chromium
```

---

## Running the Automation

To execute the automated tests and update the results in the Excel file, run:

```cmd
python IT23278844_test_automation.py --excel "IT23278844_Assignment 1 - Test cases.xlsx" --headless
```

### Parameter Reference

| Parameter | Description |
|---|---|
| `--excel` | Path to the Excel test case file |
| `--headless` | Run without a visible browser window |
| `--wait-ms` | Time (ms) to wait for output (default: 5000) |
| `--save-every` | Save the Excel file every N rows |

---

## Checking Results

After the script completes:

1. Open `IT23278844_Assignment 1 - Test cases.xlsx`.
2. The **Actual output** column will contain the text produced by the translator.
3. The **Status** column will show **FAIL** for all 50 cases, confirming the identified system weaknesses as required by the assignment.
